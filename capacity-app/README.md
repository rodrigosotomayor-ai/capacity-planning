# Capacity App

Gestión de capacity mensual para equipos PD y MKT.  
Conectado a Supabase · Desplegado en Vercel.

## Deploy en Vercel (5 min)

### Opción A — Vercel CLI (recomendado)
```bash
npm install -g vercel
vercel --prod
```

### Opción B — GitHub + Vercel Dashboard
1. Sube este repo a GitHub (público o privado)
2. Ve a vercel.com → New Project → Import Git Repository
3. Selecciona el repo → Deploy

## Desarrollo local
```bash
npm install
npm run dev
# abre http://localhost:3000
```

## Stack
- Next.js 14 (App Router)
- React 18
- Supabase (PostgreSQL)
- Tabler Icons
